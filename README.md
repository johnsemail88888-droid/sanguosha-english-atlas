# 三国杀英文图鉴

面向持有中文实体牌的英语玩家：按中文牌图找到牌，再阅读完整英文效果、大白话打法、使用时机、实战例子与常见误区。

## 收录范围

- 三国杀移动版：标准武将、界限突破、风、火、林、山、阴、雷及对应的十二名经典神将（112 名武将）
- 标准版与军争篇常用基本牌、锦囊牌、装备牌（43 种游戏牌）
- 不收录谋、势、魔等现代付费/超模扩展

## 开发与打包

```bash
npm install
npm run validate:data
npm test
npm run build
npm run pack:mac
```
